Welcome to this guide on getting started with Astro! Astro is a modern static site generator that helps you build faster websites.

## What is Astro?

Astro is an all-in-one web framework for building fast, content-focused websites. Some key features include:

- Zero JavaScript by default
- Component-based architecture
- Built-in Markdown support
- Excellent developer experience

## Quick Start

To create a new Astro project:

```bash
npm create astro@latest
```

That's it! Follow the prompts and you'll have a new Astro project ready to go.
